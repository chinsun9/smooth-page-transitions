## 실행

### 필요 종속성

- parcel ; 이 프로젝트 package.json에 포함되어 있지 않아 실행해 보려면 각자 수동 설치 필요

### 커맨드

```sh
yarn

parcel index.html
```

## setup

```cmd terminal
npm init -y
npm i -g parcel-bundler

npm i gsap
npm i @dogstudio/highway
npm i @dogstudio/highway
```

## command

```
parcel index.html

parcel build index.html --out-dir docs --public-url ./
```

- 깃허브 페이지로 배포할 때, 위 옵션을 넣어서 parcel build
